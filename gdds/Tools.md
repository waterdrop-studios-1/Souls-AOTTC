# How tools work
Crafting Progression:
1. Workbench
- Place 4 Solid Sticks.
- Place 1 Plank
- Level 0 bench
- Add teired hammers to upgrade

2. Toolsbench
- Place 4 Solid Sticks and 1 Plank
- Add teired pickaxe to upgrade

3. Refinary Table
- Place Steel Table
- Add water bottle (for rinsing)
- Add stainer (for water purification)
- Add buddy medkit (for buddy medicines)
- Add medkit (for human medicines)

4. Armor Smithing Table 
- Place 4 Solid Sticks
- Place 1 Steel Plate
- Add teired anvil for upgrade
